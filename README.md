### A propos

Ingenieur Logiciel diplome de Sorbonne Universite. Je concois des pipelines de donnees cloud-native et des systemes ML a l'intersection du genie logiciel et de l'intelligence artificielle.

Derniere experience : **Renault Group (Ampere)** -- pipeline ETL sur GCP traitant 130 000+ fichiers d'essais securite batterie, application full-stack adoptee a 100% par les ingenieurs cibles, prototype computer vision YOLOv8.

Actuellement freelance, je construis des outils d'audit de code par IA (LangChain, RAG) et une plateforme d'analyse telemetrique F1 (PySpark, Databricks, XGBoost).
